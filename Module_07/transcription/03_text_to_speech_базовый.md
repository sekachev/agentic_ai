# Text-to-Speech: Базовый уровень (Edge TTS)

## Что такое TTS

**Text-to-Speech (TTS)** — технология синтеза речи из текста. Противоположность STT (Speech-to-Text).

## Эволюция TTS

### Первое поколение

Ранние модели имели:
- Фиксированный набор голосов (мужской, женский, различные тембры)
- Актеры озвучки начитывали большие объемы текста
- Модель обучалась воспроизводить их голос
- Проблема: монотонность, отсутствие эмоций

### Современное поколение

Новые модели позволяют:
- **Выбрать базовый голос**
- **Описать стиль промптом:** "читай с французским акцентом", "саркастично", "с паузами"
- **Вставлять метки:** `[смех]`, `[вздох]`, `[пауза 2 сек]`
- **Управлять эмоциями:** грустно, радостно, взволнованно, спокойно

**Результат:** практически неотличимая от человеческой речь.

## Edge TTS: Бесплатное решение от Microsoft

### Как это работает

Microsoft встроил TTS в браузер **Edge** для озвучки веб-страниц. Open-source сообщество:
1. Изучило, как работает API
2. Создало пакет `edge-tts` для Python
3. Теперь любой может использовать бесплатно

### Установка

```bash
pip install edge-tts
```

### Использование

#### Командная строка

```bash
# Посмотреть список голосов
edge-tts --list-voices

# Создать аудио
edge-tts -v ru-RU-DmitryNeural -t "Вот что я хочу тебе сказать: генерация голоса — это просто. Как раз-два-три." -w test.mp3
```

#### Python скрипт

```python
import edge_tts

# Генерация аудио
text = "Вот что я хочу тебе сказать: генерация голоса — это просто."
voice = "ru-RU-DmitryNeural"  # или ru-RU-SvetlanaNeural

# Сохранение в файл
edge_tts.Communicate(text, voice).save("output.mp3")
```

### Доступные голоса

Для русского языка:
- **ru-RU-DmitryNeural** — мужской голос
- **ru-RU-SvetlanaNeural** — женский голос

Плюс мультиязычные голоса, которые тоже говорят по-русски.

Всего в библиотеке **сотни голосов** на десятках языков.

### Качество

По соотношению **цена/качество** (бесплатно!) конкурентов нет:
- Понятная речь
- Правильные знаки препинания
- Адекватная интонация
- Подходит для большинства задач

> "Если это бесплатно, и оно говорит понятно, со знаками препинания — что с этим может конкурировать?" 

## Практическое применение

### Озвучка книг

**Важное ограничение:** нельзя отправлять целую книгу за раз!

Модель обрабатывает текст с коэффициентом ~1.2:
- 1 минута аудио = 1.5 минуты генерации
- Длинные тексты могут упереться в лимиты сервера

**Решение:** использовать специализированные репозитории.

#### GitHub репозиторий для озвучки книг

Поиск: `edge-tts book github`

Эти инструменты:
- Разбивают книгу на абзацы
- Отправляют запросы по частям
- Используют 10 параллельных воркеров
- Склеивают результат
- Поддерживают формат **ePub**

### Где брать книги

**Flibusta.is** — библиотека бесплатных книг в формате ePub

> Можно создать сервис с бесплатными аудиокнигами. Главное — разместить в нейтральной юрисдикции.

### YouTube-контент

Целая индустрия генерации контента:
- **Анекдоты** — просто текст + озвучка + смех на фоне (4000-6000€/месяц)
- **Истории на ночь** — миллионы просмотров
- **Сказки для детей** — популярный формат
- **Исторические факты** — образовательный контент

Пример: дедушка 60 лет вечером вместо телевизора генерирует 15-секундные shorts с анекдотами и зарабатывает 4000€/месяц.

## Ограничения

### Числительные и падежи

Проблемные места для русского языка:
- "Родился в **тысяча восемьсот двадцать восьмом** году"
- "С **пятисотрублевой** купюрой"
- Склонение числительных

**Решение:** 
- Предварительная обработка текста языковой моделью
- Замена "1828" на "тысяча восемьсот двадцать восемь"
- Это дешевле, чем использовать дорогой TTS с поддержкой числительных

### Специальные термины

Пример: авиационные термины (ILS, TCAS) произносятся одинаково каждый раз, без естественных вариаций живой речи.

Можно заметить ИИ только по таким микронюансам.

## Когда использовать Edge TTS

✅ **Подходит для:**
- Озвучки статей и книг для личного использования
- Создания подкастов и YouTube-контента
- Прототипирования голосовых интерфейсов
- Образовательного контента
- Автоматизации озвучки новостей

❌ **Не подходит для:**
- Коммерческой озвучки высокого класса
- Проектов, требующих эмоциональной игры
- Случаев, когда нужен конкретный уникальный голос
