## Общая часть (модули 1-6)

### Модуль 1: Введение в AI-технологии (4 часа)

- Вводный урок: знакомство с курсом, цели, структура, инструменты
- ИИ, ML, LLM и нейросети — ключевые понятия в области искусственного интеллекта (2 часа)
- Как AI-технологии меняют мир: реальные примеры и кейсы

### Модуль 2: Работа с LLM (4 часа)

- Большие языковые модели: как работают, возможности, ограничения
- Популярные модели: ChatGPT, Claude, Gemini — сравнение и выбор
- Практический опыт: генерация текста, анализ, перевод
- Использование через API

### Модуль 3: Звук и голос (4 часа)

- Технологии TTS, STT и native-audio модели
- Инструменты распознавания и синтеза голоса: ElevenLabs и др
- Интеграция с LLM
- Пактические кейсы: голосовой ассистент, телефония и др

### Модуль 4: Изображения и визуал (4 часа)

- Как работает генерация изображений: Diffusion, GAN
- Инструменты: DALL-E, Midjourney, Stable Diffusion
- Промпт-инжиниринг для визуала: описание стиля, композиции
- Практика: создание картинок для разных задач

### Модуль 5: Автоматизация n8n (4 часа)

- Концепция No-Code: триггеры, действия, workflow
- Интерфейс n8n: ноды, соединения, настройки
- Базовые сценарии: таблицы, уведомления, интеграции
- Практика: сборка первого автоматизированного процесса

### Модуль 6: Вайб-кодинг и серверы (4 часа)

- Python через LLM: как ставить задачи ИИ для генерации кода
- Базовые операции: файлы, API, обработка данных
- Что такое VPS и зачем нужен сервер для автоматизаций
- Деплой: запуск скрипта на сервере 24/7
