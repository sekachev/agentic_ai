## Общая часть (модули 1-7)

### Модуль 1: Введение и Основы ИИ (4 часа)
- ИИ, ML, LLM и нейросети — ключевые понятия.
- Как работают веса и архитектура Transformer. 
- Информационные революции: почему именно сейчас?

### Модуль 2: Профессиональная работа с LLM (4 часа)
- Работа через API (OpenRouter, Google AI Studio).
- Структурированные данные (JSON) и язык разметки (Markdown).
- Промпт-инжиниринг: от Persona до Few-shot.
- Экономика токенов: как оптимизировать расходы.

### Модуль 3: Архитектура AI Агентов (4 часа)
- Анатомия агента: восприятие, принятие решений, инструменты.
- Паттерны проектирования: ReAct (Reason + Act), CoT (Chain of Thought).
- no-code vs code
- **Блиц-практика**: сборка агента за 1 час (n8n и Python).

### Модуль 4: No-Code vs Vibe Coding (4 часа)
- **Погружение в n8n**: создание сложных автоматизаций без кода.
- **Vibe Coding на Python**: генерация серверной логики через LLM.
- Сравнение подходов: где быстрее собрать, а где легче масштабировать.

### Модуль 5: Productization: Frontend & DevOps (4 часа)
- **Frontend через Vite + TypeScript**: современный веб-интерфейс для вашего агента.
- Мгновенный запуск кода из Google AI Studio в своем приложении.
- **Основы DevOps**: работа с VPS, установка Docker, деплой проекта.
- Запуск автономного агента на сервере 24/7.

### Модуль 6: Мультимодальность: Звук и Голос (4 часа)
- Голосовые технологии: TTS (синтез) и STT (распознавание).
- Клонирование голоса и интеграция в агентские системы.
- Практика: создание голосового ассистента.

### Модуль 7: Мультимодальность: Vision и Визуал (4 часа)
- Vision-модели: как ИИ видит и анализирует реальный мир.
- Генерация изображений (Diffusion, DALL-E).
- Итоговая сборка мультисенсорного агента (слышит, видит, говорит).
