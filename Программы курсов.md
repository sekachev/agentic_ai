## Общая часть (модули 1-7)

### Модуль 1: Введение и Основы ИИ (4 часа)
- ИИ, ML, LLM и нейросети — ключевые понятия.
- Как работают веса и архитектура Transformer. 
- Информационные революции: почему именно сейчас?

### Модуль 2: Профессиональная работа с LLM (4 часа)
- Работа через API (OpenRouter, Google AI Studio).
- Структурированные данные (JSON) и язык разметки (Markdown).
- Промпт-инжиниринг: от Persona до Few-shot.
- Экономика токенов: как оптимизировать расходы.

### Модуль 3: Архитектура AI Агентов (4 часа)
- Анатомия агента: переход от ограничений "чистой" LLM и WebUI к автономности.
- **Инструментарий и знания**: использование внешних инструментов (Skills/Tools) и RAG (Knowledge Retrieval).
- Логика мышления: паттерны ReAct, CoT, Планирование и Рефлексия.
- **Надежность и контроль**: обработка ошибок (Exception Handling) и участие человека (Human-in-the-Loop).
- **Интероперабельность**: Model Context Protocol (MCP) и взаимодействие агентов (a2a).
- Инженерный выбор: сравнительный анализ No-code и Code архитектур.
- **Блиц-практика**: параллельная сборка агента на n8n и Python (сравнение в реальном времени).

### Модуль 4: Архитектура реальных AI-сервисов (Best Practices) (4 часа)
- **Frontend vs Backend в AI**: Разделение ответственности.
- **Разбор реальных кейсов**:
    - `edusport.neuro.ee` — Спортивные тренировки и запись.
    - `ruttu.ee` — Умный дневник поездок (SaaS). Локации, отчетность, трекинг.
    - `ai.evc.ee` — Анализ фото вен и консультация.
    - `dunven.neuro.ee` — Работа с документами (JPG -> XLS).
    - `dims.neuro.ee` — разметка базы данных
    - `ruttu.ai` — AI портал для больших данных
- **Architectural Traceing**: разбор архитектуры
- **Практика**: Проектирование и сборка аналога (MVP).

### Модуль 5: Деплоймент AI-агентов (4 часа)
- **Инфраструктура**: Ubuntu Terminal (шпаргалка), навигация и управление процессами.
- **Сети и Прокси**: IP, DNS, настройка Nginx как Reverse Proxy и SSL (Certbot).
- **Frontend**: Деплой статических файлов через Cockpit Navigator и настройка Base URL.
- **Backend**: Создание API на FastAPI, запуск через Uvicorn и настройка бессмертных служб (Systemd).
- **Практика**: Развертывание связки Frontend + Backend на личном VPS.

### Модуль 6: Мультимодальность: Vision и OCR (4 часа)
- Vision-модели: как ИИ видит и анализирует реальный мир.
- Генерация изображений (Diffusion, DALL-E).
- Практика: создание визуального анализатора.

### Модуль 7: Мультимодальность: Звук и Голос (4 часа)
- Голосовые технологии: TTS (синтез) и STT (распознавание).
- Клонирование голоса и интеграция в агентские системы.
- Итоговая сборка мультисенсорного агента (слышит, видит, говорит).
