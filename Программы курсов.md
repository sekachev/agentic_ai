## Общая часть (модули 1-6)

### Модуль 1: Введение и Основы ИИ (4 часа)
- ИИ, ML, LLM и нейросети — ключевые понятия.
- Как работают веса и архитектура Transformer. 
- Информационные революции: почему именно сейчас?

### Модуль 2: Профессиональная работа с LLM (4 часа)
- Работа через API (OpenRouter, Google AI Studio).
- Структурированные данные (JSON) и язык разметки (Markdown).
- Промпт-инжиниринг: от Persona до Few-shot.
- Экономика токенов: как оптимизировать расходы.

### Модуль 3: Основы AI Агентов (4 часа)
- Анатомия агента: восприятие, принятие решений, инструменты.
- Паттерны проектирования: ReAct (Reason + Act), CoT (Chain of Thought).
- Стратегии планирования и рефлексия (самопроверка модели).
- Веб-интерфейс vs Автономный агент.

### Модуль 4: Автоматизация Agentic Workflows в n8n (4 часа)
- Концепция No-Code агентов: триггеры, действия, узлы.
- Интеграция LLM в воркфлоу: создание цепочек рассуждений.
- Использование JSON Schema для написания сценариев через ИИ.
- Практика: создание агента по сбору и анализу новостей.

### Модуль 5: Вайб-кодинг и автономные скрипты (4 часа)
- Python через призму LLM: как писать код, не будучи программистом.
- Работа с файлами, API и базами данных.
- Подготовка и настройка VPS (сервера) для работы ИИ 24/7.
- Инструменты кодинг-агентов (Antigravity и др.).

### Модуль 6: Мультимодальные системы (Звук + Визуал) (4 часа)
- Голосовые технологии: TTS (синтез) и STT (распознавание).
- Генерация и анализ изображений (Diffusion, DALL-E, Vision).
- Создание мультимодального ассистента: слышит, видит, говорит.
