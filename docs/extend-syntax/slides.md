# Source: https://mszturc.github.io/obsidian-advanced-slides/extend-syntax/slides

To create a horizontal slide use three dashes surrounded by two blank lines. To create vertical slides use two dashes surrounded by two blank lines:

```
# Slide 1
---
# Slide 2.1
--
# Slide 2.2
```


`separator`

or `verticalSeparator`

property in frontmatter
