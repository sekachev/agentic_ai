> СУТЬ:
ОТ КОДА К НАМЕРЕНИЮ.
Мы переходим от написания инструкций (как делать)
к описанию целей (что получить).

> ПОЧЕМУ ЭТО ВАЖНО (WHY IT MATTERS):
 1. Абстракция: Каждый шаг эволюции скрывает сложность.
 2. Доступность: Программирование становится доступно всем.
 3. Детерминизм: Software 1.0 предсказуем. Software 3.0 — вероятностен.

---

**ПОГРУЖЕНИЕ:**

Андрей Карпаты (бывший директор AI в Tesla) ввел понятие **Software 2.0**. Мы расширяем это до 3.0.

*   **Software 1.0 (Classic Code):**
    Программист пишет явные инструкции: `if x > 0 then y`. Это чистая логика. Python, C++, Java. Мы контролируем каждый бит.
    *Минус:* Сложно описывать нечеткие задачи (распознать лицо).

*   **Software 2.0 (Neural Networks):**
    Программист пишет архитектуру и дает данные. Веса (weights) подбираются сами в процессе обучения. Это "черный ящик".
    *Минус:* Требует огромных датасетов и мощностей.

*   **Software 3.0 (Agentic AI):**
    Программист пишет промпт (намерение) и дает инструменты. Агент сам пишет код (1.0) или использует модель (2.0) для решения.
    *Плюс:* Естественный язык как интерфейс.

```text
      УРОВЕНЬ        ИНСТРУМЕНТ      РЕЗУЛЬТАТ
    +-----------+  +-------------+  +------------+
    | SW 3.0    |  | Prompt      |  | Agent      |
    | (Agents)  |  | (English)   |  | Workflow   |
    +-----------+  +-------------+  +------------+
          |               |
          v               v
    +-----------+  +-------------+  +------------+
    | SW 2.0    |  | Data + Arch |  | Weights    |
    | (ML/DL)   |  | (PyTorch)   |  | Model      |
    +-----------+  +-------------+  +------------+
          |               |
          v               v
    +-----------+  +-------------+  +------------+
    | SW 1.0    |  | Code        |  | Binary     |
    | (Classic) |  | (Python/C)  |  | Program    |
    +-----------+  +-------------+  +------------+
```

**Сдвиг парадигмы:**
Раньше мы были архитекторами зданий (Software 1.0). Теперь мы — мэры городов (Software 3.0), которые ставят задачи службам, но не кладут каждый кирпич сами.
