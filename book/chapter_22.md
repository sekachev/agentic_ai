// CHAPTER 22 ___________________________________________
// FROM LLM TO AGENT

> СУТЬ:
ВЫХОД ИЗ БАНКИ.
LLM — это мозг в колбе, отрезанный от мира.
Агент — это киборг, подключенный к интернету и терминалу.

> ПОЧЕМУ ЭТО ВАЖНО (WHY IT MATTERS):
 1. Память: LLM не помнит прошлый диалог (Stateless). Агент помнит (Stateful).
 2. Факты: LLM выдумывает. Агент может проверить в Google.
 3. Действие: LLM выдает текст. Агент выдает результат (файл, запись в БД).

---

**ПОГРУЖЕНИЕ:**

"Чистая" LLM страдает от **сенсорной депривации**. Она обучалась на данных до 2023 года (или 2024), но она не знает, какая погода сегодня в Таллинне. Если вы спросите её об этом, она либо честно скажет "не знаю", либо начнет галлюцинировать.

Чтобы превратить модель в агента, мы даем ей протезы:
1.  **Краткосрочная память** (история чата).
2.  **Долгосрочная память** (векторная база данных / RAG).
3.  **Инструменты** (функции Python, API).

Агент — это не новая модель. Это **архитектурная обвязка** вокруг той же самой модели.

```text
       PURE LLM (Chat)          AGENTIC SYSTEM
     .-----------------.      .------------------.
     |                 |      |                  |
     |   [ ВХОД ]      |      |   [ ЗАДАЧА ]     |
     |      |          |      |       |          |
     |      v          |      |       v          |
     |   (Black Box)   |      |   (Plan & Loop)  |
     |      |          |      |    /    |    \   |
     |      v          |      |   v     v     v  |
     |   [ ТЕКСТ ]     |      | [Tool][Mem][Code]|
     |                 |      |       |          |
     '-----------------'      |       v          |
                              |   [ РЕЗУЛЬТАТ ]  |
                              '------------------'
```

**Главный инсайт:**
LLM — это всего лишь компонент. Как процессор в компьютере. Процессор без жесткого диска и экрана бесполезен для пользователя. Так же и LLM бесполезна для бизнеса без агентской архитектуры.
