# ГЛАВА 11

#### [Левая страница]

```text
// CHAPTER 11 ___________________________________________
// THE TOKEN

> СУТЬ (STATUS):
АТОМ СМЫСЛА.
Это не буква и не слово. Это цифровой обломок текста,
превращенный в число, понятное машине.

> ПОЧЕМУ ЭТО ВАЖНО (WHY IT MATTERS):
 1. Экономика: Вы платите API не за слова, а за токены.
 2. Математика: Нейросеть не умеет читать, она умеет
    считать. Токен — это способ превратить "Привет"
    в цифру "4815".
 3. Ограничения: "Контекстное окно" измеряется в них.
    Если токены кончились, модель забывает начало фразы.
```

#### [Правая страница]

**ПОГРУЖЕНИЕ:**

Представьте, что вы учите иностранный язык, но вам запретили учить буквы. Вы можете запоминать только слоги или целые короткие слова.
Слово «Машина» для компьютера — слишком длинное. Буква «М» — слишком бессмысленная.
Компьютер нарезает текст на кусочки — **токены**.

*   Частое слово «Привет» = 1 токен.
*   Редкое слово «Синхрофазотрон» = 4–5 токенов (Син-хро-фазо-трон).

**Главная ловушка:**
Токен ≠ Слово.
Токен ≠ Буква.

В английском языке 1000 токенов — это примерно 750 слов. В русском, из-за окончаний и суффиксов, расход выше: 1000 токенов — это около 400–500 слов. Нейросеть видит текст не как плавную реку, а как дискретный набор кирпичей.

**Баг восприятия:**
Когда вы спрашиваете «Сколько букв в слове lollypop?», модель часто ошибается.
Почему? Потому что для неё "lollypop" — это один цельный кирпич с ID `9912`. Она не видит швов внутри него. Она не знает, что он состоит из `l`, `o`, `l`... Она выучила этот кирпич целиком.

> **Правило:**
> Нейросеть предсказывает не *букву*, а следующий *токен*. Она играет в лего, а не в каллиграфию.

```text
       ВХОДНЫЕ ДАННЫЕ (INPUT)
      +----------------------+
      | "Learning is fun"    |
      +----------------------+
                 |
                 v
        ТОКЕНИЗАТОР (КЛИНОК)
      / / / / / / / / / / / /
                 |
                 v
      +-------+-------+------+
      | Learn |  ing  |  is  | ...
      +-------+-------+------+
      | 3402  |  581  |  11  | <--- ID Токенов
      +-------+-------+------+
         ^       ^       ^
      Корень   Суффикс  Связка


  ЧЕЛОВЕК ВИДИТ:    [ L o l l y p o p ]
                       | | | | | | | |

  LLM ВИДИТ:        [   ID: 9912      ]
```
